Ryan Berroa Cloudflare Pages final site

Upload steps
1. Unzip this package
2. In Cloudflare Pages create or update the project
3. Upload the extracted folder contents as a static site
4. No build command is needed
5. Output directory should be /

Included assets
- index.html
- styles.css
- assets/headshot.jpeg
- assets/logo.png
- assets/RBERROA-Resume_Final.pdf
- assets/Cloud.mp4
- assets/Family_Dinner_Fun_Video_Generated.mp4

Video behavior
- Videos are embedded without audio controls
- Videos autoplay muted, loop, and play inline


SEO additions included
- robots.txt
- sitemap.xml
- canonical tag
- robots meta tag
- Open Graph tags
- Twitter card tags
- JSON-LD structured data for WebSite and Person/ProfilePage
- .well-known/security.txt

After upload
1. Confirm https://ryanberroa.com/robots.txt loads
2. Confirm https://ryanberroa.com/sitemap.xml loads
3. In Google Search Console submit sitemap.xml
4. Request indexing for https://ryanberroa.com/
